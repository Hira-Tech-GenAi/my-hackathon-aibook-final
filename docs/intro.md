# AI-Driven Spec-to-Book Generator
**Zero manual writing · Fully AI-generated book from a single spec**

- Claude 3 Opus generates entire book structure & content
- Gemini 2.5 Flash creates images & code snippets
- Real-time RAG Chatbot (OpenAI + Qdrant) embedded in the book
- Beautiful dark-mode Docusaurus design (live now)
- 100% free deployment on GitHub Pages

- Qdrant Cloud (free tier) + OpenAI API
